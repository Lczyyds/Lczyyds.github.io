mysql 简单命令

启动服务器
net start mysql
停止服务器
net stop mysql

如何通过ip地址访问数据库

将mysql/表/user/Host的localhost改为“%”；
重启mysql：计算机–》管理–》服务和应用程序–》服务–》mysql–》重新启动
或者在cmd使用命令
net start mysql
net stop mysql

如何通过C#（.NET）进行数据库的连接

  1. 连接数据库
  2. 新建数据库
    数据库名：**+db
    字符集：utf8mb4
  3. 新建表
    设置字段为不可重复
    在索引中选择字段，设置索引类型为UNIQUE。
